🛡 Sécurité & conservation légale

Cette application ne nécessite ni compte ni mot de passe : toutes vos données restent exclusivement sur cet appareil, dans ce navigateur. Rien n'est envoyé sur internet.
Conséquence importante : si vous videz le cache du navigateur, changez d'appareil, ou en cas de panne, les données peuvent être perdues sans préavis. La sauvegarde régulière (export JSON ci-dessus) est donc votre seule protection.

Pour rappel, la loi française impose la conservation des factures pendant 10 ans (Code de commerce, art. L123-22). Conservez vos exports JSON dans un endroit sûr et dupliqué (cloud personnel, clé USB, etc.).

Dernière sauvegarde exportée : jamais

Crée automatiquement dans le catalogue tous les articles présents dans les factures (ventes et achats) qui n'y figurent pas encore. Utile lors d'une migration ou si des articles ont été saisis directement dans les lignes de facture sans passer par le catalogue.

🗑 Zone de danger — Réinitialisation sélective

Sélectionnez les données à supprimer. Cette action est irréversible. Exportez une sauvegarde JSON avant de continuer.
